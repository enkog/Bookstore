# Bookstore

The Bookstore is a website that allows a user to;

- Display a list of books.
- Add a book.
- Remove a selected book.


## Built With
- JS
- CSS
- JSX
- React

## Getting Started

### Prerequisites

- Node.js
- npm

### Setup

Clone the project:

```
git clone https://github.com/enkog/Bookstore.git
```

### Run project

Install dependencies:

```
npm install
npm start
```
